# excalidraw-layers

## What is this

This is an experimental tool to view layers of
[Excalidraw](https://excalidraw.com) drawings.

## How to use it

Just visit <https://dai-shi.github.io/excalidraw-layers>

and enter the link in the field and click "View Layers!"
